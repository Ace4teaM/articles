Choosing an Editor I don't recommend any WYSIWYG editor for building websites. I don't care who made it, Microsoft, Mozilla, Adobe or any other software provider, if it's WYSIWYG it's JUNK! If you paid $1400 for a Visual or WYSIWYG, you still need to learn HTML code, just to fix the problems your editor will create. If you want to use a WYSIWYG, learn to build web pages with a simple text or HTML editor first. Then you'll be equipped to fix the CRAP your WYSIWYG generates!
